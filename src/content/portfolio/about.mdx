복잡한 요구사항 속에서 **진짜 문제를 먼저 재정의**합니다. 공공기관 수탁 프로젝트들에서 그대로 개발하는 대신, 기획자·원청과 기대를 조율하며 더 나은 타협점을 제안해 프로젝트의 낭비를 줄여왔습니다.

디자인이 목적에 맞지 않다고 판단되면 디자이너와 토의를 거쳐 **직접 리디자인을 주도**합니다. UI/UX를 심미성을 넘어선 문제 해결의 영역으로 바라보며, 모든 이해관계자에게 최선의 가치를 전달하는 결과를 만드는 데 집중합니다.

서비스의 소유자가 아닌 **책임자의 스탠스**로, 독단보다 소통을 통해 팀이 같은 방향을 바라보게 합니다. 본 포트폴리오에 담긴 6개의 서비스는 기획, 개발, 배포, 운영까지 전체 과정을 직접 참여한 프로젝트입니다.
